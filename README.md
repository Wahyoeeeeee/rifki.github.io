# Rifki Wahyu Hartadi - Portfolio Website

Website portofolio pribadi yang memuat informasi CV, keahlian manufaktur (Operator Produksi, 5R, K3, AutoCAD), serta proyek Sistem Informasi.

## 🚀 Fitur
- **Responsive Design**: Tampilan rapi dan optimal di HP maupun PC.
- **Showcase Kompetensi**: Menampilkan pengalaman kerja di industri manufaktur, operasional, hingga IT.
- **Data Driven**: Konten portofolio bisa diatur dengan mudah melalui file `script.js`.

## 🛠️ Teknologi yang Digunakan
- **HTML5** & **CSS3**
- **JavaScript (Vanilla)**
- **Vercel** (untuk Hosting)
