# Rifki Wahyu Hartadi - Portfolio Website

Website portofolio pribadi yang berisi informasi CV, keahlian, dan proyek-proyek yang telah dikerjakan.

## 🚀 Fitur
- **Responsive Design**: Tampilan optimal di HP maupun Laptop.
- **Project Showcase**: Daftar proyek desain, video, dan teknologi.
- **Data Driven**: Konten diatur melalui file `script.js`.

## 🛠️ Teknologi yang Digunakan
- **HTML5** & **CSS3**
- **JavaScript (Vanilla)**
- **GitHub Pages** (untuk Hosting)
